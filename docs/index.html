instacart-order-history

instacart-order-history is a command-line tool for exporting your Instacart order history.

What does instacart-order-history do?

instacart-order-history exports your Instacart order history to JSON, CSV, or text. instacart-order-history outputs order dates, totals, retailers, and full item details.

How do I install instacart-order-history?

You install instacart-order-history with Homebrew:

brew tap mike-vincent/instacart-order-history
brew install instacart-order-history

How do I get my session cookie?

You get the session cookie from your browser. Log into instacart.com, open DevTools, go to Application → Cookies, and copy the value of _instacart_session_id.

How do I use instacart-order-history?

You use instacart-order-history by running it with your session cookie:

instacart-order-history --instacart-session-id "YOUR_SESSION_ID"

What are some instacart-order-history example commands?

# Export all orders as JSON
instacart-order-history --instacart-session-id "abc123..."

# Export last 6 months to CSV
instacart-order-history --instacart-session-id "abc123..." --months 6 -f csv

# Export to file
instacart-order-history --instacart-session-id "abc123..." -o orders.json
