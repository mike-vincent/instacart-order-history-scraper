instacart-orders

Export your Instacart order history with full item details.

No dependencies. Just Python 3.

Install

Homebrew (macOS/Linux)

brew tap mike-vincent/instacart-order-history-tool
brew install instacart-orders

Direct download

sudo curl -o /usr/local/bin/instacart-orders https://raw.githubusercontent.com/mike-vincent/instacart-order-history-tool/master/instacart-orders
sudo chmod +x /usr/local/bin/instacart-orders

Get your session cookie

  1. Log into instacart.com
  2. Open DevTools (Cmd+Option+I on Mac, F12 on Windows)
  3. Go to Application → Cookies → instacart.com
  4. Copy the value of _instacart_session_id

Usage

instacart-orders --session "YOUR_COOKIE"

Options:

--session, -s    Session cookie (required)
--months N       Last N months
--days N         Last N days
--since DATE     Since YYYY-MM-DD
--format FMT     json, csv, or text
-o FILE          Output to file
-q               Quiet mode

Output

JSON, CSV, or text with: order ID, date, total, retailer, and item details (name, quantity, size).
