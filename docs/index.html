instacart-orders

Fetch your Instacart order history. One thing, done well.

Export your complete Instacart order history with full item details to JSON, CSV, or text. No dependencies—just Python 3.

Install

Homebrew

brew tap mike-vincent/tools
brew install instacart-orders

curl

curl -o /usr/local/bin/instacart-orders \
  https://raw.githubusercontent.com/mike-vincent/instacart-order-history-tool/master/instacart-orders
chmod +x /usr/local/bin/instacart-orders

git

git clone https://github.com/mike-vincent/instacart-order-history-tool.git
cd instacart-order-history-tool
./instacart-orders --help

Get Your Session Cookie

  1. Log into instacart.com
  2. Open DevTools (Cmd+Option+I)
  3. Application → Cookies → instacart.com
  4. Copy the value of _instacart_session_id

Usage

instacart-orders --session "YOUR_COOKIE"

Options

--months N Last N months

--days N Last N days

--since YYYY-MM-DD Since date

--format json|csv|text Output format

-o FILE Output file

-q Quiet mode

Output

Orders with full item details: name, quantity, size, retailer, product ID.
